QuickQWERTY 0.1.0

By Susam Pal on 09 May 2009

Version 0.1.0 of QuickQWERTY, a touch typing tutor that runs in the web browser, is now available. This is the first release with the complete set of 21 units with a total of 101 lessons.

The lesson progression has also been improved. A new lesson is now suggested only when the current one is completed at a speed of at least 50 characters per minute.
