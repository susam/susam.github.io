MathB 0.1.0

By Susam Pal on 02 Dec 2013

MathB is a web-based mathematics pastebin service. It was developed on Sunday, 25 March 2022, as a weekend project to share mathematical notes with friends. After more than one and a half years of development and tweaks, this is the first stable release of the software. To read more about this project, please see github.com/susam/mathb.
