MathB 0.2.0

By Susam Pal on 28 Nov 2020

MathB 0.2.0 is a minor update of MathB, a web-based mathematics pastebin service. The key change in this release is the removal of image support. Image tags are now disallowed in new posts and older posts containing images will no longer be rendered. This update aims to mitigate spam on the live instances of this software.

To read more about this project, please see github.com/susam/mathb.

See the file CHANGES.md for a detailed changelog.
