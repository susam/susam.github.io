Susam Pal wrote:

@kingsfoil, thank you for leaving your comment. The guestbook comment form is implemented as a server-side program using Common Lisp and Hunchentoot. It accepts POST requests and writes each comment to a text file on the web server. I review new comments roughly once a week, delete the spam and add the genuine comments to my website as content data. A static site generator, also implemented in Common Lisp, then renders the comments along with the rest of the website.
